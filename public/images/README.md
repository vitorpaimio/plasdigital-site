# Diretório de Imagens

Coloque as fotos do projeto aqui.

## Como usar
Em projetos Vite, arquivos na pasta `public` são servidos na raiz.

Exemplo:
1. Salve sua imagem em: `public/images/minha-foto.jpg`
2. No componente React (ex: `Hero.tsx`), use:
   ```tsx
   <img src="/images/minha-foto.jpg" alt="Descrição" />
   ```
